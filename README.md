# Sing Me A Song

Back-end aplicattion for a web app that recommend a song

## Getting started

### Setting up a project

* Clone this repository: `git clone https://github.com/jefranca/Sing_Me_A_Song.git`
* Move into the project directory: `cd Sing_Me_A_Song`
* Install the dependencies: `npm i`

### Working on the project
* Run the development task: `npm start`
* Create the database using the dump.sql
* Create your .env
    * Starts a server running at http://localhost:4000
    * Automatically restarts when any of your files change
